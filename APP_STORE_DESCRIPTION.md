## Full Description

### StickyToDo - Task Management That Respects You

**Two modes. One system. Your data.**

StickyToDo is a revolutionary task management app for macOS that gives you the power of Getting Things Done (GTD) methodology combined with the visual clarity of Miro-style boards—all while keeping your data in plain-text markdown files that you own forever.

### Work Your Way

**List View** - Traditional GTD perspectives for processing tasks
• Inbox, Next Actions, Waiting For, Someday/Maybe
• Custom perspectives with complex filtering
• Keyboard-first workflow for power users
• Group by project, context, or priority

**Board View** - Visual workspace for planning and brainstorming
• Freeform canvas with infinite space
• Kanban boards for workflow management
• Grid boards for structured organization
• 60 FPS performance with 100+ tasks

### Your Data, Your Format

Every task is a plain-text markdown file with YAML frontmatter. This means:
• Edit in VS Code, Obsidian, vim, or any text editor
• Version control with git
• Sync with Dropbox, iCloud Drive, or any file sync service
• No proprietary formats or vendor lock-in
• Your data is readable and accessible forever

### Powerful Features

**Quick Capture** - Global hotkey (⌘⇧Space) with natural language parsing
`Call John @phone #Website !high tomorrow //30m`
→ Creates task with all metadata automatically extracted

**21+ Advanced Features**:
• Recurring tasks with flexible patterns
• Subtasks and hierarchies
• Full-text search with boolean operators
• Time tracking and analytics
• Task templates for common workflows
• Automation rules engine (11 triggers, 13 actions)
• Calendar integration (two-way sync)
• Siri Shortcuts (12 voice commands)
• Local notifications with interactive actions
• Weekly review workflow
• Tags with colors and icons
• Activity log and change history
• Export to 7 formats (JSON, CSV, Markdown, HTML, iCal, Things, OmniFocus)

### Designed for macOS

• Native macOS design language
• Dark mode support
• Retina-optimized graphics
• 80+ keyboard shortcuts
• Accessibility support
• Privacy-focused (all data local)
• No account required
• No internet connection needed

## Privacy Policy Summary

**Data Collection**: None
- StickyToDo does not collect, transmit, or store any personal information
- All data remains local on your device
- No analytics, telemetry, or tracking
- No account creation required
- No internet connection needed

**Optional Integrations**:
- Calendar integration uses Apple's EventKit (data stays local)
- Siri Shortcuts uses Apple's Intents framework (data stays local)
- File sync (Dropbox, iCloud) is user-controlled and optional

**Third-Party Libraries**:
- Yams (open-source YAML parser, runs locally)
