## Customization

### How to Customize Shortcuts

1. Open **System Settings** → **Keyboard** → **Keyboard Shortcuts**
2. Select **App Shortcuts**
3. Click **+** to add new shortcut
4. Choose **StickyToDo** from app list
5. Enter exact menu title
6. Press desired key combination
7. Click **Add**

### Recommended Custom Shortcuts

Add these for even faster workflow:

| Menu Item | Suggested Shortcut | Benefit |
|-----------|-------------------|---------|
| "Weekly Review" | `⌘⇧W` | Quick access |
| "Clear Completed" | `⌘⇧⌫` | Bulk cleanup |
| "Export Markdown" | `⌘⇧E` | Easy export |
| "Sync Now" | `⌘⇧S` | Force sync |

### Tips for Memorization

1. **Use Mnemonics** - ⌘N for **N**ew, ⌘F for **F**ind
2. **Consistent Modifiers** - ⇧ for variations, ⌥ for alternative actions
3. **Practice Daily** - Use shortcuts intentionally
4. **Print Cheat Sheet** - Keep visible until memorized
5. **Start with Top 10** - Master these first:
   - `⌘⇧Space` - Quick Capture
   - `⌘N` - New Task
   - `⌘F` - Search
   - `Space` - Complete
   - `⌘I` - Inspector
   - `⌘1-7` - Perspectives
   - `⌘Delete` - Delete
   - `⌘D` - Duplicate
   - `⌘K` - Complete
   - `⌘⌥1-5` - Status changes

## Pro Tips

### Workflow Combinations

**Inbox Processing:**
1. `⌘1` (Go to Inbox)
2. `↓` (Select first task)
3. `Return` (Edit title)
4. `Tab` to context, enter `@office`
5. `Tab` to project, enter `#Website`
6. `⌘⌥2` (Move to Next Actions)
7. `↓` (Next task)

**Quick Task Entry:**
1. `⌘⇧Space` (Quick Capture)
2. Type: `Call John @phone #Sales !high tomorrow //30m`
3. `Return` (Create)

**Board Workflow:**
1. `⌘⌥B` (Show Boards)
2. Click board
3. `⌘N` (New task on board)
4. `Space + Drag` (Position on canvas)
5. `⌘I` (Add details)

### Speed Techniques

**Keyboard-Only Task Creation:**
```
⌘N → Type title → Return → ⌘I → Tab through fields → ⌘Return
```

**Rapid Status Changes:**
```
Select task → ⌘⌥2 (Next Action) → ↓ → ⌘⌥4 (Someday) → ↓ → ⌘⌥1 (Inbox)
```

**Batch Flagging:**
```
⌘A → ⌘⇧F (Flag all) → Or select individually with ⌘Click → ⌘⇧F
```

## Troubleshooting

### Shortcut Not Working

1. **Check conflicts** - System Settings → Keyboard → Shortcuts
2. **Restart StickyToDo** - Some shortcuts load on startup
3. **Check modifiers** - Ensure ⌘, ⌥, ⇧, ⌃ are correct
4. **Context matters** - Some shortcuts only work in specific views
5. **Full screen** - Some macOS shortcuts disabled in full screen

### Restore Defaults

Settings → Keyboard → Reset to Default Shortcuts
